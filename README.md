# 礼泉翻鸡计分器

纯前端麻将计分器，支持礼泉翻鸡（无红版）规则。

## 使用

1. 用浏览器直接打开 `index.html`，或通过本地 HTTP 服务访问：
   ```bash
   python3 -m http.server 9876
   ```
   然后访问 http://localhost:9876

2. 选择人数（4-8 人），输入初始积分（默认 20）和玩家昵称，点击「开始」
3. 每局选择胡牌或流局；若为多人轮换场景，先勾选本局上场的 4 人，再填写表单并点击「结算」
4. 有人积分 ≤ 0 时触发大局结算，可查看详情并开始新大局
5. 历史记录中点击任意一局可查看该局积分明细
6. 数据自动保存在 localStorage，刷新页面可恢复

## 功能

- **4-8 人记分**：支持多人轮换，本局固定选择 4 人上场
- **初始积分**：每人默认 20 分，可自定义
- **大局结算**：有人 ≤ 0 分时结束本大局，显示结算后重置
- **积分详情**：每局结算后展示基础分、番型、鸡分等明细；历史记录可点击查看

## 规则摘要

- 基础分 2 分，番型叠加（边张/卡张 3、147/258/369 4、碰碰/清一色/七对/金钩吊 5、清七对/龙七对/青龙七对 7）
- 天缺：有天缺者从无天缺者每人收 2 分；缺门罚分每张 -1
- 点炮：点炮者付基础+番型；自摸：其余玩家各付全部分数（无分摊）
- 鸡分：听牌且有鸡牌者，向其余每人各收一份；金鸡（1 条）×2
- 流局：未听牌者向每位听牌者付 2 分
